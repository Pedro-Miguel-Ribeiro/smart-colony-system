# Smart Colony System

## Objetivo

Sistema inteligente para gerenciamento energético de uma colônia.

## Funcionalidades

- Organização de dados
- Tomada de decisão
- Previsão de energia
- Análise de consumo

## Tecnologias

- Python

## Como executar

Execute o comando abaixo no terminal:

python main.py

## Exemplo de saída

=== SISTEMA DA COLÔNIA ===
SUGESTÃO: armazenar energia excedente
MODO ECONOMIA ATIVADO
Previsão de energia eólica: 27
